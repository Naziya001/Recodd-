# Recodd- A Commission Free Freelancing Platform
RECODD is a commission-free freelancing platform that connects businesses and freelancers directly. It helps freelancers earn fairly without high commission charges and allows businesses to hire skilled professionals at affordable costs.
The platform includes features like project posting, freelancer portfolios, secure payments, direct communication, and admin management. It is designed using modern web technologies such as React.js, Node.js, MySQL/MongoDB, Firebase, and Razorpay/Stripe.
